Functional extensionality

In most of this formalization, we assume functional extensionality. For that, we use the following abbreviation.
Definition funext
           {A : Type}
           {B : A → Type}
           {f g : ∀ (a : A), B a}
           (p : ∀ (a : A), f a = g a)
  : f = g.
Proof.
  apply functional_extensionality_dep.
  exact p.
Qed.
